# Muniverse

An interactive web experience where users can feel and manipulate music visually by dragging floating planets (tokens) into the earth (music controller) This project explores how audio, motion, and interaction design can work together to create a more immersive way to experience music.

Instead of traditional controls (buttons/sliders), users interact with sound through:

Upload and play audio files
Spatial movement interactions
Visual feedback
Users can drag floating tokens into the blob to modify sound:

Volume up / down
Speed up / slow down
Echo effect
Low-pass filter (deeper sound)
High-pass filter (sharper sound)
Next track
Undo last effect
Tech Stack: React (Vite) Web Audio API JavaScript (ES6+) CSS (inline styles + animations)

Future Improvements:

Spotify / external playlist integration
More advanced audio visualization
Mobile interaction optimization
Expanded visual feedback system
